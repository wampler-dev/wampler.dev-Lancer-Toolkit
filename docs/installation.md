# Installation & Initial Setup

Follow this document to set up a packet-based Lancer mission using the OSR toolkit.

Prerequisites:
- Foundry VTT v12+
- Lancer system
- Lancer Mission Journal Generator v9 (CLEAN)

Steps:
1. Create or open a Lancer world.
2. Run the Mission Journal Generator to create mission folders.
3. Create packet journals (B##, C##, D##).
4. Paste approved macros into Foundry.
5. Set the initial packet and build the dashboard.

If anything goes wrong, run Packet Audit & Repair.
